현재 브랜치의 PR을 찾아 논의 내용을 코멘트로 게시한다.

## 실행 절차

1. 현재 브랜치의 PR 번호 확인:
   ```bash
   gh pr view --json number,title,url
   ```

2. 이 대화에서 나눈 코드리뷰 논의 내용을 아래 형식으로 정리하여 PR 코멘트 게시:
   ```bash
   gh pr comment {PR_NUMBER} --body "..."
   ```

## 코멘트 형식

```markdown
## 코드리뷰 논의 요약

**논의 주체:** 사용자 ↔ 오케스트레이터(Claude)

### 논의 내용
{사용자와 오케스트레이터가 나눈 리뷰 항목별 질의응답 정리}

### 결정 사항
{논의를 통해 확정된 변경 사항 또는 유지 결정 목록}

### 적용 여부
- 작업 전 실행 시: ⏳ 작업 예정
- 작업 후 실행 시: ✅ 적용 완료 / ⏭️ 향후 과제로 이관
```

## 규칙
- 논의 주체를 항상 명시한다 ("사용자와 오케스트레이터 간 논의")
- 결정 사항은 근거와 함께 작성한다
- 모든 내용은 한국어로 작성한다
